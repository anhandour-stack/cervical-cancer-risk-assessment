# Cervical Cancer Risk Assessment

Application d'aide à la décision médicale pour l'évaluation du risque de cancer du col de l'utérus.

## 📋 Description
      Ce projet développe un outil permettant aux médecins d'évaluer le risque de cancer du col de l'utérus à partir des données médicales des patientes.

  ## 🏗️ Structure du projet
├── notebooks/ # Analyses exploratoires (EDA)
├── src/ # Code source du pipeline ML
├── app/ # Interface utilisateur (Streamlit)
├── tests/ # Tests automatisés
├── .github/workflows/ # CI/CD avec GitHub Actions
├── data/ # Données (brutes et traitées)
├── models/ # Modèles entraînés
└── README.md # Documentation
